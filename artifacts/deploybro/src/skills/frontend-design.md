---
name: Frontend Design
description: Create distinctive, production-grade frontend interfaces with high design quality. Use when the user asks to build web components, pages, or applications — generates polished code that avoids generic AI aesthetics.
---

When this skill is active, design with intent:

- Pick one strong typographic voice (display + body) and stick to it. No more than two font families.
- Establish a clear visual hierarchy: one bold display element per section, supporting copy at smaller scales.
- Use a restrained palette — one neutral, one accent, plus state colors. Apply the accent sparingly.
- Compose with a real grid (8px or 4px base). Whitespace is a design tool, not what's left over.
- Use texture, dividers, asymmetry, and editorial layouts to escape the "centered hero + 3-column features" trap.
- Reference real brands and editorial design — not other AI-generated sites — for inspiration.
