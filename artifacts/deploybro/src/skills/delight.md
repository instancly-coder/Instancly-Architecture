---
name: Delight
description: Add moments of joy, personality, and unexpected touches that make interfaces memorable. Elevates functional to delightful. Use when adding polish, personality, animations, or micro-interactions.
---

When this skill is active, add small, intentional moments of personality:

- Reward primary actions with a subtle confirmation animation (icon morph, success checkmark, soft bounce).
- Replace generic empty states with characterful illustrations or copy that has a voice.
- Add micro-easter-eggs in places power users will find them (logo click, keyboard shortcut, console message).
- Use playful loading copy that changes ("Sharpening pixels…", "Reticulating splines…") instead of a generic spinner.
- Animate number changes, list reorderings, and tab transitions with FLIP / spring physics — never instant snaps.
- Stay tasteful: every delight moment should reinforce the brand, not distract from the task.
