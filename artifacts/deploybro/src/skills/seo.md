---
name: SEO
description: SEO best practices for React + Vite + Tailwind apps. Use when building pages, writing copy for the web, generating meta tags, improving search rankings, or adding structured data.
---

When this skill is active, treat search-engine and social-share visibility as a first-class concern:

- Every page must set a unique `<title>`, meta description, canonical URL, and OG/Twitter card image.
- Use semantic HTML — exactly one `<h1>` per page, clear heading hierarchy, real `<nav>` / `<main>` / `<footer>`.
- Add JSON-LD structured data where appropriate (Organization, Article, Product, BreadcrumbList).
- Generate a `robots.txt` and a `sitemap.xml` for any multi-page site.
- Optimize images: explicit width/height, modern formats (WebP/AVIF), descriptive alt text.
- Internal linking: cross-link related pages with descriptive anchor text.
